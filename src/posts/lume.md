Lume est un générateur de sites statique rapide et flexible pour Deno

- Il supporte n'importe quel moteur de template
Créez des pages en utilisant Markdown, Nunjucks, Liquid, JSX, TSX, JavaScript, TypeScript, Pug, Eta... ou ajoutez facilement votre propre moteur.
- Vous pouvez stocker les données dans votre format préféré
Stockez vos données dans des formats statiques comme JSON ou YAML. Utilisez JavaScript ou TypeScript pour obtenir les données d'une base de données ou d'une API
- Lume permet de rraiter les pages et les ressources HTML
Les processeurs peuvent compiler et optimiser des ressources telles que CSS ou JavaScript. Ils peuvent également transformer le code HTML en utilisant l'API DOM.
- Vous pouvez exécuter vos scripts et écouter les événements
Vous pouvez créer des scripts personnalisés comme dans NPM et les exécuter à partir du CLI ou après un événement quelconque.
- Et tout cela avec une API propre et intuitive
Configurez la construction de votre site dans un seul fichier `_config.ts` ou `_config.js` avec des [plugins](https://lume.land/plugins/?status=all) et une API simple et propre.
- Déployer n'importe où
Les sites statiques peuvent être hébergés (gratuitement) n'importe où GitHub/GitLab Pages, Deno Deploy, Vercel, Netlify... [Explorez les possibilités de déploiement](https://lume.land/docs/advanced/deployment/)

## Comment l'installer ?

Il vous suffit d’aller sur ce site et de suivre l’instruction donné. Et puis vous pouvez codé votre site et le mettre en ligne.
